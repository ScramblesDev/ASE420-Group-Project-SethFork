Tetris game with the following features:

 -Sound effects
 -Piece Saving
 -Piece preview
 -Dark Mode
 -Color Palettes
 -Game Over Screen
 -Ability to pause
 -Speed Increase
 -Score Keeper

The main file is refactored_tetris.py

The features to complement refactored_tetris.py are located in the src directory with tests being located in the tests directory
